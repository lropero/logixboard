# logixboard
Freight Forwarder Engagement Solution

<img src="https://github.com/lropero/logixboard/blob/master/logixboard.gif?raw=true" width="640">

### Requires
- Node v12.14.1

### Installation
```sh
$ npm run install:all
```

### Configuration
- Set googleMapsApiKey in `src/config.js`

### Usage
```sh
$ npm run start
```
